Software & Coding

This site was created with the latest version of Jekyll. A Ruby script was used to fetch all the book metadata from Amazon.

HTML5 markup is used all throughout.

Design

This site was designed in collaboration with Nimbupani Designs.

The primary goal of the design process was to keep things minimal. However, special attention was paid to the typography.

Design inspiration came from the following sources:

  • The Elements of Typographic Style by Robert Bringhurst
  • Administrative Debris by Ryan Tomayko
  • Dive into Mark by Mark Pilgrim
