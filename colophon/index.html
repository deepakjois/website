Software & Coding

This site was created with the latest version of Jekyll. A Ruby script was used to fetch all the book metadata from Amazon.

HTML5 markup is used all throughout.

Design

This site was designed in collaboration with Nimbupani Designs.

The primary goal of the design process was to keep things minimal. However, special attention was paid to the typography because I was inspired after reading The Elements of Typographic Style.

More design inspiration came from the following sources:

  • Administrative Debris
  • Dive into Mark
