Software & Coding

This site was created with the latest version of Jekyll. The website codebase is hosted in Github too.

HTML5 markup is used throughout.

Design

This site was designed in collaboration with Nimbupani Designs.The primary goal of the design process was to keep things minimal. However, special attention was paid to the typography.

Design inspiration came from the following sources:

  • The Elements of Typographic Style by Robert Bringhurst
  • Administrative Debris by Ryan Tomayko
  • Dive into Mark by Mark Pilgrim

Some things I incorporated in the typography, which might be of interest :

  • Vertical Rhythm
  • Use the best available ampersand
  • Typographic Marks Unknown (from where I picked up the hedera between the menu items)
