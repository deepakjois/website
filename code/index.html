Software

gchartrb

This project is in hibernation. You can fork the code on GitHub and do what you like with it.

gchartrb is a Ruby wrapper around the Google Chart API. It is inspired by another Ruby based graphing library called Gruff. Visit the website for more information.

Tabbie

Aditya Krishnan and me were the original creators of Tabbie, an open source Tab Software for British Parliamentary style debating tournaments. It was first used in NTU Worlds 2004.

Apart from NTU Worlds 2004, Tabbie has been used in many tournaments around the world, including two World Universities Debating tournaments in Vancouver (2006) and Bangkok (2008).

Klaas Van Schelven is the current maintainer for Tabbie.

Bits of Code

Structure and Interpretation of Computer Programs (Exercises)

SICP is a great book. Peter Norvig writes in his review of the book

To use an analogy, if SICP were about automobiles, it would be for the person who wants to know how cars work, how they are built, and how one might design fuel-efficient, safe, reliable vehicles for the 21st century. The people who hate SICP are the ones who just want to know how to drive their car on the highway, just like everyone else.

I started reading SICP and decided to use Haskell to do the exercises. I am putting up my solutions on github.

Project Euler Problems

I am using Project Euler to learn Haskell. I have put up some problem solutions on github.

TwitterArchiver

TwitterArchiver is a script written in Haskell. It archives all your past Tweets in a JSON file. It also has the ability to sync an already existing archive and update it with the latest feeds.

PyFlickrUploadr

This code is no longer being maintained.

PyFlickrUploadr is a Python script to upload photos to Flickr from the commandline. It has a variety of things you can set from the command-line like tags, privacy level etc.

Gnome Wallpaper Changer

This code is no longer being maintained.

A long time ago I wrote a script to change/cycle wallpapers automatically in Gnome on Ubuntu Linux. I put up the code on a post on my old blog. A couple of people even contributed a few variants to it.
